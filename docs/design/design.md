# 謎解きアプリ
## 画面一覧
| 画面名 | MVPに入れるか |
|---|---|
| Top画面| ⚪︎ |
| 謎解き一覧画面 | ⚪︎ |
| 設定画面| x |
| マイページ画面| x |
| ランキング画面<br>→Topに表示しててもいいかも| x |
| ログイン画面| x |
| ゲーム画面| ⚪︎ |

## 想定される動き方
- ログインしててもしていなくてもゲームができる
- Top画面から謎解き一覧画面に遷移してやりたい謎を選ぶ
- 謎解きが終わったら謎解き一覧画面に戻る

## 機能
- Top画面は各画面へ遷移するのみ
- 謎解き一覧画面は途中の状態を保存しておける
    - 保存できる数は3ゲームくらいにする
