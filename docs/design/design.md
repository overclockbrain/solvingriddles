# 謎解きアプリ
## 画面構成
- Top画面
    - 遷移できる画面一覧
        - 謎解き一覧画面
        - 設定画面
        - マイページ画面
        - ランキング画面→Topに表示しててもいいかも
        - ログイン画面
- 謎解き一覧画面
    - 繊維できる画面一覧
        - Top画面
        - 設定画面
        - マイページ画面
        - ランキング画面→Topに表示しててもいいかも
        - ログイン画面
        - ゲーム画面

## 想定される動き方
- ログインしててもしていなくてもゲームができる
- Top画面から謎解き一覧画面に遷移してやりたい謎を選ぶ
- 謎解きが終わったら謎解き一覧画面に戻る

## 機能
- Top画面は各画面へ遷移するのみ
- 謎解き一覧画面は途中の状態を保存しておける
    - 保存できる数は3ゲームくらいにする
