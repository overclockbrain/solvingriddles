# 🕵️‍♂️ 謎解きアプリ (Mystery App)

Spring Bootの学習と復職リハビリを兼ねて開発中の、Webブラウザで遊べる謎解きゲームです。
アーケードゲームのように、サクッと遊んでスコアを競えるシステムを目指しています。

## 🛠 技術スタック
- **Language:** Java 21
- **Framework:** Spring Boot 3
- **Template Engine:** Thymeleaf
- **Infrastructure:** Docker / Render
- **Design:** Mermaid (画面遷移図など)

## 📏 開発ルール (Commit Guidelines)
コミットメッセージは **Conventional Commits** に基づき、以下のプレフィックスを付けて統一します。

**フォーマット:** `prefix: コミット内容`

| Prefix | 意味 | 使いどき |
| :--- | :--- | :--- |
| **feat** | ✨ 新機能 | 新しい画面や謎解き機能を追加した時 |
| **fix** | 🐛 バグ修正 | 動かないコードやバグを直した時 |
| **docs** | 📚 ドキュメント | READMEや画面遷移図(designフォルダ)を更新した時 |
| **style** | 🎨 見た目 | CSS修正やコードのフォーマット整形（動作に影響しないもの） |
| **refactor** | ♻️ リファクタ | 機能を変えずにコードを綺麗にした時 |
| **test** | 🧪 テスト | JUnitなどのテストコードを追加・修正した時 |
| **chore** | 🔧 雑用 | Dockerfileの設定やライブラリ更新など |

### コミット例
- `feat: 第1問の画面を作成`
- `docs: 画面遷移図を更新`
- `fix: 誤字を修正`
細かい変更など記述したいことがあれば一つ空行を入れて記載するようお願いします。

## 📂 ディレクトリ構成
- `src/`: ソースコード本体
- `docs/`: 設計図やタスク管理ドキュメント
    - `design/`: 画面遷移図など
    - `todo/`: タスクリスト
